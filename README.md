## 진행 방식

1. 매주 일요일 밤 8시에 디스코드에서 코드 리뷰

   - 문제를 못 풀었을 경우에는 어디서 막혔는지, 어느 부분이 이해가 안가는지 피드백 주고 받기

   - ⭕ - 다른 사람의 소스 코드를 참고하지 않고 스스로 잘 풀었을 때
